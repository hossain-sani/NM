# Numerical Methods Lab - MATLAB 🚀

Welcome to my Numerical Methods repository! This project contains a collection of MATLAB scripts and functions developed during my Lab sessions. The focus is on implementing mathematical algorithms to solve problems that are difficult or impossible to solve analytically.

---

## 📂 Contents

The repository is organized by the type of numerical problem:

### 1. Root Finding Algorithms
*   **Bisection Method**: Finding roots by repeatedly bisecting an interval.
*   **Newton-Raphson Method**: Using derivatives for faster convergence.
*   **Regula Falsi**: The False Position method.

### 2. Linear Algebraic Equations
*   **Gauss Elimination**: Solving systems of equations via row reduction.
*   **Gauss-Seidel**: An iterative method for large sparse systems.
*   **LU Decomposition**: Factoring a matrix into Lower and Upper triangular matrices.

### 3. Interpolation & Curve Fitting
*   **Lagrange Interpolation**: Polynomial fitting for a given set of points.
*   **Newton’s Divided Difference**: Constructing interpolating polynomials.
*   **Least Squares Regression**: Best-fit line for scattered data.

### 4. Numerical Integration & Differentiation
*   **Trapezoidal Rule**: Estimating the area under a curve.
*   **Simpson’s 1/3 and 3/8 Rules**: Higher-order integration techniques.

### 5. Ordinary Differential Equations (ODEs)
*   **Euler’s Method**: Basic first-order numerical procedure.
*   **Runge-Kutta (RK4)**: A high-accuracy 4th-order method for solving ODEs.
